## 学期总览

第一学期主要建立编程基础和数据科学基础，为后续的高级课程做准备。

### 重点学习领域

1. **编程技能** - Python和Java编程
2. **系统知识** - Unix/Linux系统操作
3. **数据处理** - 数据分析和可视化
4. **机器学习** - 基础算法和深度学习
5. **学术技能** - 研究方法和学术写作

### 课程关系

```mermaid
graph TD
    A[Programming Basics] --> B[Data Structures]
    B --> C[Data Science]
    C --> D[Machine Learning]
    A --> E[Database Systems]
    E --> C
    F[Research Skills] --> G[Academic Writing]
```

课程之间存在递进关系，编程基础支撑数据科学学习，数据科学为机器学习提供基础。
