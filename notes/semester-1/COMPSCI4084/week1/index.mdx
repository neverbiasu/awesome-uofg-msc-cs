## 概要
- 本周采用 tAPP + Lab 的形式, 聚焦 Python 入门与开发环境搭建。
- 主题覆盖：Python 基础语法, 条件分支, OOP 入门；Git/GitHub 协作；Conda 环境与（可选）Colab。

## 本周安排（按顺序）
| Session | 主题 | 说明 |
|---|---|---|
| Class 0 | 课程介绍；Git/GitHub；Conda 与 Colab | 建立环境与工具链, 熟悉协作流程 |
| Class 1 | Python 基础与条件分支；OOP 初步 | 变量/类型/表达式/条件；类与对象, 方法, 简单封装 |

## 学习目标
1. 完成开发环境搭建（Anaconda/Conda + VS Code；Colab 可选）。
2. 掌握 Git 基本操作（clone/commit/push/branch/PR）, 理解团队协作流程。
3. 夯实 Python 基础（变量, 类型, I/O, 条件与布尔逻辑）。
4. 理解 OOP 基础概念（类, 对象, 方法, 封装）。

## 关键知识点
- 环境与工具：
  - Conda 环境创建/激活/依赖导出；避免与系统 Python 混用。
  - Git 基本工作流与提交信息规范；远程协作与冲突处理入门。
- Python 基础：
  - 数据类型, 运算符, 输入/输出, 字符串处理；缩进与代码风格。
  - 条件判断（if/elif/else）, 逻辑运算（and/or/not）, 三元表达式。
- OOP 入门：
  - 类定义, 构造器, 实例属性与方法；简单封装与 __str__/__repr__。

## 实践清单（自检）
- [ ] 创建并激活课程专用 Conda 环境（确认解释器与包版本一致）。
- [ ] 初始化 Git 仓库, 设置 user.name/user.email, 完成首次 commit 与 push。
- [ ] 编写一个包含条件分支的小程序（例如成绩转等级）。
- [ ] 定义一个简单类（如 Account/Student）, 实现属性与方法并打印实例信息。
