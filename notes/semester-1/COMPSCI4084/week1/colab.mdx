## 关键知识点

### 1. 快速上手（Getting Started）
- 核心概念：打开 Colab、创建新笔记本、运行首个 Python/Markdown 单元。
- 步骤
  1) 打开网址：colab.research.google.com。
  2) 新建笔记本并命名为“Introduction to Google Colab”。
  3) 添加代码单元并运行：
```python
print("Hello, Colab!")
```
  4) 添加 Markdown 单元，简要说明本笔记本用途。
- 注意事项/最佳实践：首个代码单元建议放置“依赖安装”，保证一键运行可复现。

---

### 2. 启用硬件加速与绘图（GPU、NumPy、Matplotlib）
- 步骤
  1) 菜单 Runtime → Change runtime type → Hardware accelerator 选择 GPU（或 TPU）。
  2) 安装依赖并执行简单计算与绘图：
```bash
pip -q install numpy matplotlib
```
```python
import numpy as np
import matplotlib.pyplot as plt

arr = np.random.rand(100) * 10
plt.plot(arr)
plt.title("Random * 10")
plt.show()
```
- 最佳实践：固定关键库版本（如 `numpy==1.26.*`），降低环境漂移；绘图前确保已 `import matplotlib.pyplot as plt`。

---

### 3. 协作与修订历史（Share & Revision History）
- 步骤
  1) 右上角“Share”设置共享邮箱与权限。
  2) 双方在不同单元协作编辑与运行。
  3) 通过“File → Revision history”查看与回溯更改。

---

### 4. Drive 挂载与文件 IO
```python
from google.colab import drive
drive.mount('/content/drive')

base = '/content/drive/MyDrive'
import os; os.listdir(base)

with open(f"{base}/colab-demo.txt", "w") as f:
    f.write("Saved from Colab.")
```
> 会话结束临时文件会丢失；关键数据落盘至 Drive 或远端存储。

---

### 5. 在 Colab 使用 R（最小示例）
```bash
apt-get -y update && apt-get -y install r-base
R -e "install.packages('IRkernel'); IRkernel::installspec(user = FALSE)"
```
- 切换：Runtime → Change runtime type → 选择 R → Save。
- R 示例：
```r
print("Hello, R in Colab!")

data(mtcars)
head(mtcars)
```
