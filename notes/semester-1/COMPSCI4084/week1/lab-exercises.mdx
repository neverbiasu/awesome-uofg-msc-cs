## 概要
- 目标：将 Git/Conda 与 Python 基础综合练习，打通从环境到代码到协作的闭环。
- 适用：为 Week 2 考核做准备，形成可复现的实践套路。

## 学习目标
- 通过动手实践熟悉工具链与基本语法，形成可复现开发流程。

## 任务清单
### 1. Git/GitHub（Practical Exercises）
- 初始化仓库并推送到远端；创建 feature 分支；提交并发起 PR。
- 制造并解决一个轻量冲突，记录解决步骤与结论。

### 2. Conda 环境（Practical Exercises）
- 创建 `progsd` 环境并安装核心依赖；导出 environment.yml；
- 在 VS Code 选择解释器并运行一个最小脚本。

### 3. Python 基础与决策（Lab Exercises）
- 编写“成绩到等级”程序：输入校验、边界测试、对齐输出。
- 扩展：支持批量处理并输出汇总统计（均值/最大/最小）。

## 进阶挑战（可选）
- 实现一个命令行工具：
  - `grades --input file.csv --pass 50`，输出统计报告与排名前 N。
  - 结构化输出（JSON/CSV 二选一），确保可重定向到文件。

## 提交建议
- 小步提交，信息清晰；`.gitignore` 忽略环境与临时文件。
- 在 README 记录环境与运行方法，确保可复现。

## 自我检查
- [ ] 是否从零到 PR 完成一次闭环并解决过冲突？
- [ ] 是否能在干净环境按 README 成功运行你的程序？
- [ ] 程序是否覆盖非法输入与边界值，输出是否对齐/可读？
