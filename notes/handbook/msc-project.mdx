# 📌 表4：MSc项目阶段要求

| **💻 MSc项目阶段要求** | **📌 核心要点速览** |
|------------------------|---------------------|
| **项目性质** | 🔬 **CS+为研究型**（论文为主，无答辩） |
| **时间地点** | 🗓️ **12周**（6月-9月）<br/>📍 **必须在校**！远程→影响学生签证⚠️ |
| **成绩要求** | ✅ **≥D3** 才能拿学位<br/>❌ E1或以下 → 可修改重交，**封顶D3** |
| **导师沟通** | 👨‍🏫 **定期会面** + **持续进展**<br/>（出勤/进度将被监控） |
| **关键提醒** | 🚫 不接受全职工作/实习理由的EC申请 |

## 🎯 项目成功要素

### 📋 规划阶段（S2期间）
- **选题确定**：与导师充分讨论研究方向
- **文献调研**：建立扎实的理论基础
- **方法设计**：明确研究方法和技术路线

### 🏃‍♂️ 执行阶段（S3期间）
- **定期汇报**：每周/双周与导师会面
- **进度管控**：按计划推进，及时调整
- **文档记录**：持续记录研究过程和发现

### 📝 完成阶段
- **论文撰写**：结构清晰，逻辑严谨
- **质量检查**：多轮修改和完善
- **按时提交**：严格遵守截止时间

## ⚠️ 常见陷阱

- **远程工作**：严重影响签证状态，绝对避免
- **沟通不足**：与导师保持密切联系
- **时间管理**：12周时间紧迫，需要高效利用
- **范围控制**：避免项目范围过大无法完成

## 🆘 支持资源

- **导师指导**：学术和技术支持
- **同期学生**：互相交流和支持
- **学校设施**：实验室、图书馆等资源
- **IT支持**：技术问题解决方案
