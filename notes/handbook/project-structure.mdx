# 📌 表1：项目结构与学业规划

| **🎓 项目结构与学业规划** | **📌 核心要点速览** |
|--------------------------|---------------------|
| **总学分** | **180分** = 120授课 (S1+S2) + 60项目 (S3) |
| **升学要求** | ✅ GPA ≥12.0<br/>✅ 总学分120<br/>✅ ≥90学分M级<br/>✅ ≥90学分≥D3<br/>✅ 无G/H等级 |
| **学位等级** | 🥇 **Distinction**：总GPA≥17.5 + 项目≥B1<br/>🥈 **Merit**：总GPA≥14.5 + 项目≥C1<br/>✅ **合格**：项目≥D3 |
| **退出奖项** | 🎓 **PGDip**：120学分, GPA≥9.0, ≥80学分≥D3<br/>📜 **PGCert**：60学分, GPA≥9.0, ≥40学分≥D3 |
| **学期安排** | 🗓️ S1: 9月-12月<br/>🗓️ S2: 1月-5月<br/>🗓️ S3: 6月-9月（项目期） |

## 🎯 关键要点

- **180学分**是完整硕士学位的总要求
- **GPA 12.0**是升学到项目阶段的最低门槛
- **Distinction**需要整体优秀表现，不只是高GPA
- **退出路径**为无法完成完整项目的学生提供替代选择

## 📅 时间规划建议

1. **S1阶段**：适应学习节奏，确保基础课程达标
2. **S2阶段**：冲刺高分，为整体GPA加分
3. **S3阶段**：专注项目，争取优秀等级
