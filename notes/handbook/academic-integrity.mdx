# 📌 表5：学术诚信与行为规范

| **⚠️ 学术诚信与行为规范** | **📌 核心要点速览** |
|--------------------------|---------------------|
| **抄袭后果** | 🚫 **零容忍** → 0分 / 记录在案 / 甚至开除！ |
| **代码规范** | 💻 严禁复制他人代码<br/>🚫 GitHub公开自己作业代码 = 允许他人抄袭 = 违规！ |
| **AI使用** | 🤖 **允许使用**，但**必须声明**<br/>（参考大学AI政策，保持透明+符合伦理） |
| **出勤监控** | 📱 **SafeZone App打卡**<br/>❗ 缺勤3周 → 劝退+取消签证sponsorship |
| **合同作弊** | 🚫 **严禁代写**（Freelancer/EssayBay等）→ **直接开除风险**！ |

## 🚨 高风险行为警示

### 💻 编程作业
- **直接复制代码** = 学术不端
- **公开作业代码** = 为他人提供抄袭便利
- **使用他人GitHub** = 违反原创性要求

### 📝 论文写作
- **段落级复制** = 抄袭
- **未标注引用** = 剽窃
- **购买代写服务** = 严重违规

### 🤖 AI工具使用
- **ChatGPT/Claude等** = 可以使用但必须声明
- **不声明AI使用** = 误导性行为
- **过度依赖AI** = 可能影响学习效果

## ✅ 正确做法指南

### 📚 学术诚信
1. **独立完成**：所有作业都应该是你的原创工作
2. **正确引用**：任何参考资料都要标明出处
3. **声明协助**：包括AI工具使用、同学讨论等

### 🤝 合理协作
1. **讨论思路**：可以讨论概念和方法
2. **分享资源**：可以推荐参考资料
3. **避免抄袭**：不直接分享答案或代码

### 📱 出勤管理
1. **按时打卡**：使用SafeZone App记录出勤
2. **请假手续**：生病或紧急情况及时申请EC
3. **保持记录**：维护良好的出勤记录

## 🆘 违规后果

| 违规程度 | 可能后果 |
|----------|----------|
| 轻微违规 | 警告、重做作业 |
| 中等违规 | 该课程0分、记录在案 |
| 严重违规 | 停学、开除 |
| 签证违规 | 取消sponsorship、遣返 |
