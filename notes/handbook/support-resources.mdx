## ✉️ 邮件沟通技巧

### 📝 邮件格式
```
主题：[MSc CS] 简明问题描述
发件人：学生邮箱（@student.gla.ac.uk）
内容：
- 自我介绍（姓名、学号、课程）
- 问题描述（清晰具体）
- 期望回复时间
- 感谢结尾
```

### ⏰ 回复时间预期
- **紧急问题**：当日回复
- **一般问题**：1-3工作日
- **复杂问题**：1周内
- **假期期间**：延长至2周

## 🚨 紧急情况处理

### 🆘 学术紧急情况
1. **缺考/迟交**：立即联系课程协调员
2. **健康问题**：申请EC + 医疗证明
3. **技术故障**：联系IT Services
4. **误解要求**：尽快澄清，避免恶化

### 🏥 个人紧急情况
1. **医疗急救**：999（英国急救电话）
2. **心理危机**：Samaritans 116 123
3. **校园安全**：Campus Security
4. **签证问题**：ISS紧急热线
