# 📌 表3：学业表现与评估体系

| **📊 学业表现与评估体系** | **📌 核心要点速览** |
|--------------------------|---------------------|
| **评分标准** | 🎯 **22分制**：A1=22分 → H=0分<br/>（50分=C3=12.0，及格线） |
| **GPA计算** | 📈 按课程学分加权平均<br/>（例：20学分B2=16 → 贡献 20/120×16=2.67） |
| **补考政策** | 🔄 **8月补考**，成绩**封顶C3 (12.0)**<br/>❗ **不影响** Merit/Distinction评定（只看首考） |
| **迟交惩罚** | ⏳ **-2档/天**，超5天或作业已发回 → **H (0分)** |
| **特殊情况** | 🆘 **EC申请**：因病/突发事件可申请缓考/延期（需证明） |

## ⚠️ 重要提醒

- **12.0是升学门槛**，但不是舒适区
- **补考虽有机会，但限制较多**
- **迟交惩罚严厉**，时间管理很重要
- **EC申请**需要充分证据支持
